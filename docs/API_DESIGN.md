## 1. Conventions

### 1.1 Request Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Conditional | `Bearer {accessToken}` |
| `X-Tenant-ID` | Conditional | Active organization ID; must match JWT `tid` |
| `X-Request-ID` | No | UUID; generated by gateway if omitted |
| `Content-Type` | Yes* | `application/json` (*except S3 presigned upload) |
| `Accept` | No | `application/json` |

### 1.2 Response Envelope (Success)

```json
{
  "success": true,
  "data": {},
  "meta": { "requestId": "uuid", "timestamp": "2026-06-01T12:00:00.000Z" }
}
```

### 1.3 Response Envelope (Error)

```json
{
  "success": false,
  "error": { "code": "ERROR_CODE", "message": "Human-readable message", "details": [] },
  "meta": { "requestId": "uuid", "timestamp": "2026-06-01T12:00:00.000Z" }
}
```

### 1.4 Paginated Response

```json
{
  "success": true,
  "data": [],
  "pagination": { "page": 1, "limit": 20, "total": 142, "totalPages": 8 },
  "meta": { "requestId": "uuid", "timestamp": "2026-06-01T12:00:00.000Z" }
}
```

### 1.5 Pagination Query Parameters

| Param | Type | Default | Max |
|-------|------|---------|-----|
| `page` | number | 1 | — |
| `limit` | number | 20 | 100 |
| `sort` | string | `-createdAt` | Prefix `-` for descending |

### 1.6 Role Hierarchy (for minimum role checks)

`viewer` < `business_user` < `legal_reviewer` < `tenant_admin` < `super_admin`

## 2. Authentication Reference

| Code | Name | Headers / Body | Used For |
|------|------|----------------|----------|
| `PUB` | Public | None | Register, login, password reset |
| `RT` | Refresh Token | Body: `refreshToken` | Token refresh |
| `JWT` | Bearer JWT | `Authorization: Bearer {accessToken}` | Authenticated user |
| `JWT+T` | JWT + Tenant | JWT + `X-Tenant-ID` | All tenant-scoped routes |
| `JWT+T+R` | JWT + Tenant + Role | JWT+T + minimum role | RBAC-protected routes |
| `SA` | Super Admin | JWT (`isSuperAdmin: true`) | Platform admin routes |
| `SVC` | Service Token | `X-Service-Token: {jwt}` | Internal microservice calls |
| `APIK` | API Key | `X-API-Key: {key}` + `X-Tenant-ID` | Partner API (Phase 2) |

**JWT access token claims:** `sub` (userId), `tid` (tenantId), `role`, `email`, `isSuperAdmin`, `exp` (15 min)
